## 简介

一个移动端的在线词典，提供单词搜索和单词本功能

- 使用indexedDB对数据进行持久化存储
- 实现单词文件的导入及导出
- 使用webWorker对文件进行编码和解码
- 使用虚拟列表进行优化
- 实现夜间模式的切换
